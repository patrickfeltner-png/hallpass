# HallPass

Classroom sign-out app with shared Firebase/Firestore data.

## Deploy

This is a static web app. Deploy the project root to Vercel.

Required files:

- `index.html`
- `styles.css`
- `app.js`
- `firebase-config.js`
- `vercel.json`

## Firebase

Firestore data is stored at:

```txt
schools/default-school/apps/hallpass-state
```
